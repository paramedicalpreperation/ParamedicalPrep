## Q1
Question: Who published the first human Electrocardiogram?
Option A: William Einthoven
Option B: Augustus D. Waller
Option C: William Harvey
Option D: Luigi Galvani
Correct Answer: Option B
Difficulty: Medium
Explanation: Augustus D. Waller, a British physiologist, recorded and published the first human electrocardiogram in 1887 using a capillary electrometer. William Einthoven later invented the much more accurate string galvanometer and coined the term "electrocardiogram".
## Q2
Question: Exercise ECG is a valuable procedure for the assessment of:
Option A: Ventricular Function
Option B: Valvular Function
Option C: Cardiomyopathy
Option D: Ischemic Heart Disease
Correct Answer: Option D
Difficulty: Easy
Explanation: Exercise ECG (also known as a Treadmill Test or TMT) is primarily utilized to evaluate the heart's response to stress and is a frontline non-invasive tool for diagnosing Ischemic Heart Disease (IHD).
## Q3
Question: In an ECG paper each large square equals _________ in height.
Option A: 0.04 mv
Option B: 0.1 mv
Option C: 0.5 mv
Option D: 0.20 mv
Correct Answer: Option C
Difficulty: Easy
Explanation: On standard ECG paper, the vertical axis measures amplitude (voltage). Standard calibration is 10 mm = 1 mV. Therefore, one large square (5 mm) vertically equals 0.5 mV.

## Q18

## Q48
Question: In a patient with Aortic stenosis and resting cardiac output of 5L/minute with a LV-Aortic peak to peak pressure difference of 65mm Hg, quick aortic valve area by Hakki formula in cm² is:
Option A: 0.63
Option B: 0.83
Option C: 0.93
Option D: 1.12
Correct Answer: Option A
Difficulty: Medium
Explanation: The Hakki formula is a simplified method for estimating valve area: \text{AVA} = \frac{\text{Cardiac Output (L/min)}}{\sqrt{\text{Peak-to-Peak Gradient}}}. Therefore, 5 / \sqrt{65} = 5 / 8.062 = 0.62. The closest option is 0.63.

## Q50
Question: If SVC saturation is 70% and IVC saturation is 82% the MVO2 will be:
Option A: 73%
Option B: 76%
Option C: 80%
Option D: 70%
Correct Answer: Option A
Difficulty: Medium
Explanation: Mixed Venous Oxygen Saturation (MVO2) can be estimated using Flamm's equation: \frac{3 \times \text{SVC} + 1 \times \text{IVC}}{4}. Thus, (3(70) + 82) / 4 = (210 + 82) / 4 = 292 / 4 = 73\%.
