# Paramed Prep Project Guidelines

This document defines the engineering, design, performance, UI/UX and coding standards for the Paramed Prep project.

## Vision

Build India's most trusted, fastest and most user-friendly learning platform for Paramedical students.

## Core Principles

- Learning First
- Clean UI
- No Unnecessary Distractions
- Performance Before Features
- Crash-Free Experience
- Mobile First
- Tablet Optimized
- Easy to Maintain
- Scalable Architecture

 ## Engineering Rules

- Modular Architecture
- Reusable Components
- Offline First
- Firebase Sync
- Write Once, Use Everywhere
- No Hardcoded Values
- Every Feature Must Be Easy to Modify
- Crashlytics Enabled
- Test Every Feature Before Next Feature

 ## UI & UX Rules

- Minimal and Clean Design
- CBT Inspired Mock Test Interface
- Responsive Layout
- Consistent Color Language
- Fast Navigation
- Accessibility Friendly
- Smooth Animations
- No UI Clutter
- Mobile First, Tablet Perfect

 ## Performance Rules

- App Must Feel Instant
- No ANR (Application Not Responding)
- No Memory Leaks
- Optimized Image Loading
- Smooth Scrolling
- Fast App Launch
- Offline Support
- Firebase Sync in Background
- Crash-Free User Experience
- Low Battery Consumption

 ## Git & Documentation Rules

- Every New Feature Must Have Documentation
- One Feature at a Time
- Test Before Commit
- Meaningful Commit Messages
- Keep Git History Clean
- Never Commit Broken Code
- Update CHANGELOG for Important Changes
- Documentation Before Implementation

 ## Coding Standards

- Follow MVVM Architecture
- Follow Material 3 Design Guidelines
- Use Kotlin Best Practices
- Keep Functions Small and Focused
- Use Meaningful Variable and Function Names
- Avoid Duplicate Code
- Prefer Reusable Components
- Add Comments Only Where Necessary
- Every Screen Must Be Responsive
- Every New Feature Must Be Tested Before Merge

 ## Long-Term Goal

Build a stable, scalable and trusted learning platform for Paramedical students that can easily grow from 12 courses to 40+ courses, from thousands to lakhs of questions, while remaining fast, clean and easy to maintain.
